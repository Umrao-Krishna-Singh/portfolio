My Work

Project 1

Dev Blog

Technologies used: Node.js, MongoDB, EJS, CSS, Git, Heroku, TinyMCE

A blogging website made using EJS template. The database is stored on mongodb atlas and the CRUD operations can be managed on the site without any need to touch the back-end. CSS provides presentation, layout, animations etc. It also supports rich text using TinyMCE api. This allows formatting of the blogpost while typing it out on the website. Version control is handled by git and the site is deployed as a heroku app. First load is slow, kindly give some time. The tag and description have a default value set and will be filled automatically if left blank.

Project 2

Keeper App

Technologies used: React, Node.js, CSS, git, Vercel.

A clone of google keep. It is a front-end project made using Reactjs which can store ideas (stored on client-side i.e. temporarily) and update the interface accordingly. There are few starting values coded into the app, which are stored in an array but they can also be delted by user. CSS provides layout, colours etc. Version control is done using git and the app is deployed on Vercel.
